Compact icon-only button for dense portal toolbars and table row actions.

```jsx
<IconButton label="Settings" variant="secondary"><GearIcon/></IconButton>
<IconButton label="Delete" variant="danger" size="sm"><TrashIcon/></IconButton>
```

Variants: `secondary` (default), `ghost`, `primary`, `danger`. Sizes `sm` (32px) / `md` (40px). Always pass `label` for accessibility.

Initials or image avatar; periwinkle fill by default, matching the portal "MR" brand mark.

```jsx
<Avatar name="Mountain Recruiting" square size={40} />
<Avatar name="Scott Samaniego" />
<Avatar src="/photo.jpg" name="Kayla Slaughter" size={32} />
```

Use `square` for the app/brand mark; circular for people. Override `color` for a different fill.

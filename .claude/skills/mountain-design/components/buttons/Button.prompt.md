Primary action control for Mountain Group — royal-blue fill on light/dark surfaces; use for the single most important action in a view.

```jsx
<Button variant="primary" size="lg">Book a Free Consultation</Button>
<Button variant="outline-dark" size="lg">See How It Works</Button>
<Button variant="secondary" size="sm">Edit</Button>
<Button variant="danger" size="sm">Delete</Button>
```

Variants: `primary` (royal blue, default), `secondary` (white + grey border, portal default), `ghost` (text-only blue), `danger` (red — Delete / Logout), `outline-dark` (transparent + white border, for the dark marketing hero). Sizes `sm | md | lg`. Set `pill` for fully-rounded marketing CTAs. Pass `iconLeft` / `iconRight` for icons.
